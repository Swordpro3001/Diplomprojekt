About the Project

Ai4Ads is a research project analyzing urban traffic data to detect and correct inconsistencies using machine learning. It combines scalable DevOps deployment, optimized data processing, and intelligent anomaly handling to support media planning and mobility insights.

🤖

Automated Deployment

DevOps-compliant infrastructure enables automated deployment of containerized AI models via CI/CD pipelines, ensuring scalable processing of large traffic datasets.

📊

Data Refinement

A curated dataset is created by merging backups and external sources, reduced to essential features for optimal machine learning performance and enhanced consistency.

🔍

Anomaly Detection

Machine learning identifies inconsistencies in traffic frequency across segments, factoring in POIs like hospitals, malls, and parking areas to improve detection accuracy.

🛠️

Anomaly Correction

A smoothing algorithm corrects detected anomalies without introducing new ones, ensuring data integrity for downstream AI applications.
